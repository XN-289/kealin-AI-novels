# ✦ Kealin AI Novels

**反 AI 味的 AI 小说创作工具**

一个为网文写作者设计的 AI 辅助创作系统，内置"反 AI 味引擎"，从提示词层面压制 AI 写作的典型毛病。

[快速开始](#-快速开始) · [功能特性](#-功能特性) · [反 AI 味引擎](#-反-ai-味引擎) · [支持模型](#-支持模型)

## 为什么做这个

市面上的 AI 写作工具有一个通病：**生成的内容 AI 味太重**。

"心中一凛"、"眼中闪过一丝XX"、"嘴角勾起一抹XX"、排比句、段尾总结抒情、形容词堆砌……这些是 AI 写作的指纹，读者一眼就能看出来。

本项目的核心思路：**不是让 AI 写得更多，而是让 AI 写得更像人**。

通过精心设计的提示词体系，从生成阶段就压制 AI 味，配合右键菜单的"AI 味检测"和"去 AI 味重写"功能，反复打磨直到满意。

## 🚀 快速开始

```bash
# 克隆项目
git clone git@github.com:XN-289/kealin-AI-novels.git
cd kealin-AI-novels

# 安装依赖
pip install -r requirements.txt

# 配置 API Key
cp .env.example .env
# 编辑 .env 填入你的 API Key

# 启动
python app.py
```

打开浏览器访问 `http://localhost:20000`

## ✨ 功能特性

### 创作流程

```
基础设定 → AI 生成大纲 → 章节细纲 → 逐章正文 → 右键打磨 → 完成
```

| 功能 | 说明 |
|------|------|
| 📋 **大纲生成** | 基于设定自动生成故事大纲，要求矛盾尖锐、转折有因果 |
| 📖 **章节拆分** | 大纲自动拆分为章节细纲，每章有明确事件驱动 |
| ✍️ **正文创作** | 逐章生成正文，内置反 AI 味规则 |
| 🖱️ **右键菜单** | 选中文本右键，一键检测/重写/压缩 |
| 🤖 **AI 助手** | 右侧面板，随时问问题、构思剧情 |
| 📚 **类型模板** | 预置 7 种常见网文类型（都市、仙侠、玄幻、系统流等） |
| 💾 **自动保存** | 所有内容自动保存到浏览器，支持导入导出 |
| 🔧 **运行时配置** | 在界面中直接切换模型，无需重启 |

### 右键菜单（正文区选中文本）

| 操作 | 说明 |
|------|------|
| **AI味检测** | 逐项检测 AI 痕迹，给出 0-100 分评分 |
| **去AI味重写** | 按禁用词清单 + 结构规则重写选中内容 |
| **压缩精简** | 砍废话，字数减 40% 信息量不减 |
| **砍形容词** | "温暖的阳光" → "阳光" |
| **加动作写情绪** | 把形容词情绪改成行为描写 |
| **加口语感** | 像讲故事，不像写作文 |

## 🛡️ 反 AI 味引擎

这是本项目的核心差异化能力。以下规则已内置在提示词中：

### 🚫 禁用词清单

出现以下任何一个，该段直接重写：

```
心中一凛    眼中闪过一丝XX    嘴角勾起一抹XX    一股XX涌上心头
仿佛XX一般  宛如XX            不由得              情不自禁
目光深邃    意味深长           若有所思            恍然大悟
不禁XX      这一刻他明白       他知道，XX
```

### 📐 结构规则

- **句子长短交替** — 禁止通篇 15-25 字的"标准句"
- **"的"字密度** — 每段不超过 3 个
- **禁止排比句** — "他感到…他感到…他感到…"直接扣分
- **禁止段尾总结抒情** — "这就是XX的意义啊"全部删掉

### 💬 对话规则

- 每句对话必须：暴露信息 / 暴露性格 / 推动行动（三选一）
- 不同角色说话方式必须不同
- 用动作代替"XX说道"、"XX淡淡道"
- 要有打断、跑题、答非所问

### 🎭 情绪规则

- 用动作和结果表达情绪，不要写"他很愤怒" → 写"他把杯子摔了"
- 禁止大段心理独白
- 让读者从行为中自己感受

## 🤖 支持模型

所有 OpenAI 兼容格式的 API 均可使用：

| 模型 | Endpoint | 推荐场景 |
|------|----------|----------|
| **DeepSeek** | `https://api.deepseek.com/v1/chat/completions` | 性价比高，中文能力强 |
| **通义千问** | `https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions` | 阿里系，稳定 |
| **豆包** | `https://ark.cn-beijing.volces.com/api/v3/chat/completions` | 字节系 |
| **OpenAI** | `https://api.openai.com/v1/chat/completions` | GPT-4o 效果最好 |

配置方式：启动后在界面「模型设置」中填写，或直接编辑 `.env` 文件。
