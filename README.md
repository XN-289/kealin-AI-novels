## 为什么做这个

AI 写小说，快是真快，假也是真假。

"心中一凛"、"眼中闪过一丝XX"、"嘴角勾起一抹XX"、排比三连、段尾抒情总结、形容词不要钱地堆……这些是 AI 写作的指纹，读者扫一眼就知道是机器写的。

**Kealin 的活儿就一个：把这股味洗掉。**

不是让 AI 写得更多，是让 AI 写得像人。从提示词层面就把 AI 味按住，配合右键菜单的检测和重写功能，反复打磨，直到以假乱真。

## 🚀 快速开始

```bash
# 克隆
git clone git@github.com:XN-289/kealin-AI-novels.git
cd kealin-AI-novels

# 装依赖
pip install -r requirements.txt

# 配置 API Key
cp .env.example .env
# 编辑 .env，填入你的 Key

# 启动
python app.py
```

浏览器打开 `http://localhost:20000`，开写。

## 🎯 它能干嘛

### 一条线走完

```
设定 → 大纲 → 章节细纲 → 逐章正文 → 右键打磨 → 导出
```

### 核心功能

| 干什么 | 怎么干 |
| ------ | ------ |
| 📋 **大纲** | 喂设定，出大纲。要求矛盾尖锐、转折有因果，自带黄金三章设计 |
| 📖 **拆章** | 大纲自动拆成章节细纲，每章有事件驱动、情绪曲线、钩子强度 |
| ✍️ **写正文** | 逐章生成，内置反 AI 味引擎，开头有冲突，结尾有钩子 |
| 🖱️ **右键打磨** | 选中文字右键，15 项操作一键搞定 |
| 🤖 **AI 助手** | 右侧面板随时问，自动感知当前章节上下文 |
| 📚 **类型模板** | 都市、仙侠、玄幻、系统流、末日、言情、历史、科幻、游戏、悬疑 |
| 📤 **导出** | JSON / TXT / Markdown 三格式 |
| 🔧 **换模型** | 界面里直接切，不用重启 |

### 右键菜单（15 项）

| 操作 | 干嘛用 |
| ---- | ------ |
| **AI 味检测** | 10 项维度打分，0-100，哪里假一目了然 |
| **去 AI 味重写** | 按禁用词清单 + 结构规则重洗一遍 |
| **压缩精简** | 砍废话，字数减 40% 信息量不丢 |
| **砍形容词** | "温暖的阳光" → "阳光" |
| **加动作写情绪** | "他很愤怒" → "他把杯子摔了" |
| **加口语感** | 像讲故事，不像写作文 |
| **加伏笔** | 埋线 + 标注回收点 |
| **改节奏** | 紧张处快刀斩，重要处慢镜头 |
| **加感官描写** | 视觉/听觉/嗅觉/触觉/味觉，具体细节替代抽象形容 |
| **续写** | 接着往下写 500-1000 字 |
| **改风格** | 热血 / 悬疑 / 搞笑 / 文艺 / 硬核，一键切换 |

### 其他能力

- **角色管理** — 角色卡片自动注入提示词，防 OOC
- **世界观笔记** — 设定、规则、势力统一管理，保持一致性
- **灵感笔记** — 随手记，支持标签，一键插入正文
- **拆书分析** | 粘贴参考文本，AI 拆解写作技巧
- **一致性检查** — 人名、地名、时间线、性格前后是否矛盾
- **读者视角审查** — AI 模拟读者，指出哪里会让人弃书
- **版本历史** — 自动快照，最多 20 版，随时回滚
- **批量生成** — 一键逐章生成全文，自动避限流
- **写作日历** — 每日字数可视化，激励持续输出
- **快捷键** — `Ctrl+S` 保存 · `Ctrl+E` 导出 · `Ctrl+/` 聚焦助手

## 🛡️ 反 AI 味引擎

这是 Kealin 的核心。55 轮迭代，专洗 AI 味。

### 🚫 禁用词（40+个）

出现任何一个，该段直接重写：

```
心中一凛    眼中闪过一丝XX    嘴角勾起一抹XX    一股XX涌上心头
仿佛XX一般  宛如XX            不由得              情不自禁
目光深邃    意味深长           若有所思            恍然大悟
不禁XX      这一刻他明白       他知道，XX          眉头微蹙
嘴角上扬    心中暗道           不觉间              霎时间
此刻的他    他深知             无疑                显然
毫无疑问    不言而喻           与此同时            就在这时
突然间      猛然间             刹那间              恍惚间
不知不觉    本能地             下意识地            条件反射般
如同XX一般  好似XX             恰似XX
```

### 📐 结构规则

- **长短交替** — 禁止通篇 15-25 字的标准句
- **"的"字密度** — 每段不超过 3 个
- **禁排比** — "他感到…他感到…他感到…"直接扣分
- **禁段尾抒情** — "这就是XX的意义啊"全删
- **信息密度** — 每段必须有新东西，禁止原地踏步
- **感官细节** — 具体的替代抽象的

### 💬 对话规则

- 每句对话必须：暴露信息 / 暴露性格 / 推动行动（三选一）
- 不角色说话方式必须不同
- 用动作代替"XX说道"
- 要有打断、跑题、答非所问
- 对话要有潜台词

### 🎭 情绪规则

- 用动作和结果表达，不要写"他很愤怒"
- 禁止大段心理独白
- 让读者从行为中自己感受

### 📖 开头和结尾

- **黄金 200 字** — 开头必须有冲突或悬念
- **结尾钩子** — 每章结尾必须留悬念
- **信息投喂** — 每章至少透露一个新信息

### 📊 AI 味检测（10 项维度）

禁用词 · 排比句 · 段尾抒情 · 形容词堆砌 · 对话标签 · 心理独白 · "的"字密度 · 句子长度均匀度 · 句式多样性 · 词汇多样性

## 🤖 支持模型

所有 OpenAI 兼容格式的 API 都能用：

| 模型 | 推荐理由 |
| ---- | -------- |
| **DeepSeek** | 性价比高，中文能力强 |
| **通义千问** | 阿里系，稳定 |
| **豆包** | 字节系 |
| **OpenAI** | GPT-4o 效果最好 |

启动后在界面「模型设置」里填，或直接改 `.env`。
